# Análise de Desempenho de Jogadores

Aplicação em Python desenvolvida para cadastrar jogadores, registrar o desempenho em partidas e realizar a análise de dados como total de gols e desempenho por jogo, utilizando estruturas de dados como listas e dicionários.

# Funcionalidades
- Cadastro de jogadores com nome e quantidade de partidas disputadas
- Registro da quantidade de gols em cada partida
- Cálculo automático do total de gols por jogador
- Armazenamento estruturado dos dados em listas e dicionários
- Listagem geral dos jogadores cadastrados com código identificador
- Consulta detalhada do desempenho individual de cada jogador
- Validação de entradas e controle de fluxo da aplicação

# O que é analisado
- Total de gols marcados por jogador
- Desempenho por partida
- Organização e visualização dos dados de forma estruturada
- Consulta dinâmica de dados a partir do código do jogador

## Tecnologias Utilizadas
